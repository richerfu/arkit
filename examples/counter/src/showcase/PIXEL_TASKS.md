# Pixel Tasks

以“组件 + 对应 demo 页面”为最小任务单位，目标是像素级贴近 `react-native-reusables`。

## 状态说明

- `R1 已完成`：组件与 demo 已有可运行基线，实现与参考大体一致。
- `R2 已细化`：第二轮已修尺寸、圆角、边框、间距或排版细节。
- `待设备审计`：需要在真机 / 模拟器横竖屏下继续目检确认。
