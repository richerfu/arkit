# i18n

路径：`examples/i18n`

i18n 示例展示 `.ftl` 文件和 `i18n!` 宏的配合。

## 文件结构

```text
examples/i18n/src/lib.rs
examples/i18n/locales/zh-CN.ftl
examples/i18n/locales/en-US.ftl
```

## 使用方式

1. 用宏声明 `tr` 模块。
2. 把 `tr::I18n` 放进 State。
3. 切换语言时调用 `set_locale`。
4. view 中用 `state.i18n.tr(tr::message_id(...))` 取文案。

## 开发注意

所有 locale 文件必须和 fallback 文件拥有相同 key 和变量集合。这样业务代码只会调用编译期存在的 typed function。
